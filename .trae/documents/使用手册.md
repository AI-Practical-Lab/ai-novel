# 小说写作工具使用手册（新手版，逐步详解）

> 适用对象：第一次使用本工具的作者  
> 目标：按“登录 → 新建小说 → 设定集 → 分卷/路标 → 章纲 → 人物认知 → 正文 → 优化 → 伏笔 → AI 助手 → 下载”完整闭环走通，并能稳定复用

---

## 术语速览（先读 1 分钟）

- 分卷：整本小说按“卷/部”拆分的大结构单位。
- 路标：从分卷大纲进一步拆出的可执行剧情节点，用来“生成章节”。
- 章纲（Beat Sheet）：单章的内部结构/节奏设计（目标、冲突、钩子、场景/节拍等）。
- 人物认知：角色在当前章节之前“知道/误解/目标”的信息快照，保证行为一致性。
- 伏笔：可被后续“回收”的铺垫信息条目，记录和追踪剧情承诺。
- AI 助手“@提问”：@某个对象（章节或设定）进行上下文关联的提问。
- AI 助手“指令模式”：把输入当作“可执行编辑命令”，直接产出修改结果。

## 1. 登录

**入口位置**  
打开系统首页即为登录页。

**操作步骤**  
1. 选择登录方式：
   - 密码登录：输入“账号/手机号 + 密码”，点击【登录】。
   - 验证码登录：点击顶部切换到【验证码登录】，输入“手机号 + 验证码”，点击【登录】。
2. 登录成功后，系统通常跳转到项目/编辑工作台。

**关键按钮文案**  
- 【密码登录】 / 【验证码登录】（页签/切换）
- 【登录】

**注意事项**  
- 必填项为空会导致按钮点击无反应或提示错误；补齐后重试。
- 登录后若仍停留在本页，刷新浏览器或再次点击【登录】。

## 2. 新建小说：用向导完成初步创建（4 步）

**入口位置**  
工作台顶部或侧边栏点击【新建小说】进入向导。

### Step 1：基础信息
**操作步骤**  
1. 填写标题、简介/灵感等基础信息。  
2. 如需灵感：点击【随机灵感】或【生成创意】，等待生成后可手动微调。  
3. 点击【下一步】。

**关键按钮文案**  
- 【随机灵感】 / 【生成创意】  
- 【下一步】

**注意事项**  
- 标题尽量不要留空；后续生成会以标题作为语义锚点。

### Step 2：题材与风格
**操作步骤**  
1. 选择类型（如奇幻/科幻/悬疑等）与风格（偏克制/热血/轻松等）。  
2. 点击【下一步】。

**关键按钮文案**  
- 类型/风格选项  
- 【下一步】

**注意事项**  
- 拿不准就选贴近的风格；后续还能在设定/总纲中微调。

### Step 3：核心设定（强烈建议认真填写）
**操作步骤**  
1. 填写世界观规则、主线矛盾、主角目标、限制条件（能力上限/代价/阵营关系）。  
2. 点击【下一步】。

**关键按钮文案**  
- 核心设定文本框/字段  
- 【下一步】

**注意事项**  
- 写出“可执行规则”（而非抽象形容词）能显著提升后续 AI 稳定性。

### Step 4：总纲/大纲
**操作步骤**  
1. 点击【AI 重新生成总纲】可产出整体结构。  
2. 人工检查主线/人物动机/节奏是否基本合理。  
3. 点击【完成向导/创建】。

**关键按钮文案**  
- 【AI 重新生成总纲】  
- 【完成】 / 【创建】

**注意事项**  
- 总纲不必一开始完美；先完成后续流程，再逐级细化与修正。

## 3. 设定集：世界观/人物/地点等支持 AI 优化（重点详解）

**入口位置**  
在编辑工作台左侧/侧边栏点击【设定集】，选择分类（人物/世界观/地点/势力/物品/时间线等）。

**通用操作步骤**  
1. 新建设定：选择分类 → 点击【新增】 → 填写标题与正文 → 【保存】。  
2. 编辑设定：点击条目 → 修改内容 → 【保存】。

### 3.1 AI 优化入口（通用）
**操作步骤**  
1. 进入某个设定条目详情。  
2. 点击【AI 优化/AI 润色/AI 补全】。  
3. 在“优化指令”输入框中说明你的目标与约束（例如：补全童年事件、强化矛盾点、不要改硬设定）。  
4. 点击【开始优化】并等待返回。

**关键按钮文案**  
- 【AI 优化】 / 【AI 润色】 / 【AI 补全】  
- 【开始优化】 / 【生成】  
- “优化指令”输入框

**注意事项**  
- 指令越具体、约束越明确（不改姓名/性别/关键事实），结果越可控。

### 3.2 人物设定优化（示例指令可直接复用）
**建议指令**  
- “保持姓名、年龄、身份不变。补全其童年事件与情绪触发点，强化与反派的矛盾来源，并给出 3 条典型口头禅。”  
- “将人物设定改为更适合第一人称叙事，语气更克制，不要新增无关角色。”

### 3.3 对比与选择接受（非常重要）
**操作步骤**  
1. AI 返回后会显示【原文】与【AI 修改后】两列对比。  
2. 对每个差异块，点击【接受】或【拒绝】。  
3. 如需快捷处理，使用【全部接受】或【全部拒绝】（若提供）。  
4. 点击【应用/保存】写回设定条目。

**关键按钮文案**  
- 【接受】 / 【拒绝】  
- 【全部接受】 / 【全部拒绝】  
- 【应用】 / 【保存】

**注意事项**  
- 谨慎对待“凭空新增关键设定”的改动；优先接受结构化表达与信息补全。

### 3.4 设定集 AI 审查（Review）
**作用**  
面向“全设定/某分类/某条目”的一致性与质量检查，快速发现问题并给出修复建议。

**入口位置**  
- 设定集页面顶部工具区（Lore 区域顶部工具栏）的【AI审查】按钮。  
- 点击后会切换到“审查面板”（lore-ai 标签页），展示审查建议卡片列表。

**操作步骤**  
1. 选择审查范围：  
   - 【当前条目】/【当前分类】/【全设定集】。  
2. 选择审查维度（常见）：  
   - 逻辑冲突（自相矛盾、因果链断裂）  
   - 术语一致（名词/称谓/地名/能力描述统一性）  
   - 硬设定违规（姓名/性别/身份等被不当改动）  
   - 时间线问题（事件前后顺序矛盾、年龄/年份不匹配）  
   - 角色语气偏差（说话风格与人设不符）  
   - 敏感/禁用词（如需过滤）  
3. 点击【开始审查】，等待生成报告。

**审查结果呈现**  
- 问题清单：严重程度（高/中/低）、问题描述、定位跳转、修复建议。  
- 处理方式：  
  - 在建议卡片的“明确修改”区使用【复制替换文本】按钮复制 AI 给出的修正文本；  
  - 点击【去修改】按钮，系统会定位到对应设定项（进入该条目的编辑视图）；  
  - 将复制的文本粘贴到对应字段中，按需微调后【保存】；  
  - 如需比对流程，可在该设定条目内触发【AI 优化】，进入对比界面逐条【接受/拒绝】；  
  - 【忽略】（记录为已知问题，不做修改）  
- 注：当前前端不提供【一键修复/批量应用】，请按上述“复制 → 去修改 → 粘贴保存”的手动方式逐条处理。  
- 报告导出（如提供）：可【下载审查报告】用于团队协作与留存。

**注意事项**  
- 高严重度优先：例如硬设定违规、时间线矛盾先处理。  
- 一键修复前建议进入“对比界面”逐条核验，避免误改已确认内容。  
- 审查后建议统一术语：在设定集建立“术语表/名词库”，减少未来不一致。

## 4. 分卷优化：先优化分卷大纲，再用“路标”生成章节（重点详解）

**入口位置**  
在项目导航中进入【分卷/卷管理】。

### 4.1 分卷大纲优化
**操作步骤**  
1. 打开某分卷详情，编辑标题与分卷摘要/目标。  
2. 点击【AI 优化分卷大纲】并等待结果。  
3. 进入对比界面，逐条【接受/拒绝】，最后【保存】。

**关键按钮文案**  
- 【AI 优化分卷大纲】  
- 【接受】 / 【拒绝】 / 【保存】

**注意事项**  
- 分卷大纲建议明确“开端-发展-转折-高潮-结局”与主角代价路径。

### 4.2 路标页面（核心用法）
**入口位置**  
分卷详情页或“路标”页签。

**操作步骤**  
1. 新建路标：点击【新增路标】，填写“标题 + 描述（发生了什么、谁参与、结果/后果、推动主线）”。  
2. 可补充参与人物、冲突/转折、与分卷目标关联。  
3. 生成章节：在路标项点击【生成章节】，等待系统拆出多个章节标题/摘要。  
4. 检查生成的章节顺序与过渡是否合理；不满意，返回路标加细描述后再生成。

**关键按钮文案**  
- 【新增路标】  
- 【生成章节】

**注意事项（路标写作标准）**  
- 一句话能说清“谁要做什么，为什么，遇到什么阻碍，结果如何”。  
- 少用抽象口号（如“变强”），多写可执行事件（如“偷账本失败同伴被抓”）。

## 5. 路标生成多个章节后：进入章节，先生成“章纲”

**入口位置**  
章节目录中点击某一章 → 切到【章纲/Beat Sheet】页签或模块。

**操作步骤**  
1. 点击【AI 生成章纲】并等待结果。  
2. 章纲通常包含“目标/冲突/钩子/场景节拍”。  
3. 人工微调后点击【保存章纲】。

**关键按钮文案**  
- 【AI 生成章纲】  
- 【保存章纲】

**注意事项**  
- 若本章摘要很空，先补一句“本章要解决的冲突/目标”，再生成章纲。

## 6. 初始化人物认知（保证一致性）

**入口位置**  
章节编辑页切到【人物认知】视图。

**操作步骤**  
1. 点击【初始化/生成人物认知】。  
2. 检查是否出现“提前知道真相/认知穿越”等问题。  
3. 调整后【保存】。

**关键按钮文案**  
- 【初始化/生成】  
- 【保存】

**注意事项**  
- 人物认知依赖“章纲、前文内容、出场人物”，因此第 7 步的选人很关键。

## 7. 章节正文编辑：先选择“出场人物”，再用 AI 辅助写作

### 7.1 选择本章出场人物（必做）
**入口位置**  
章节编辑页的【出场人物/选择人物】入口。

**操作步骤**  
1. 勾选本章实际出场的人物。  
2. 点击【保存/应用】。

**关键按钮文案**  
- 【选择人物】  
- 【保存/应用】

**注意事项**  
- 出场人物会影响对白风格、冲突编排、认知更新准确性。

### 7.2 AI 辅助写作
**入口位置**  
正文编辑区顶部，点击【AI 辅助写作】。

**操作步骤**  
1. 在弹窗“写作指令”中说明要生成的段落类型与约束（字数/视角/节奏）。  
2. 点击【开始生成】；生成后插入/追加到正文（以系统行为为准）。  
3. 完成后【保存章节】。

**关键按钮文案**  
- 【AI 辅助写作】  
- 【开始生成】  
- 【保存章节】

**注意事项**  
- 只写某一段时，明确“从哪里开始写/写到哪里停”；如支持插入位置，先把光标放好。

## 8. AI 辅助优化：多维度分析 → 对比选择 → 保存（重点详解）

**入口位置**  
章节编辑页顶部或右侧工具区的【AI 优化/分析与优化】。

**操作步骤**  
1. 选择优化维度（逻辑一致性/人物一致性/节奏张力/语言表达/伏笔机会等）或一键优化。  
2. 等待 AI 返回“优化后版本”。  
3. 进入对比界面，逐块【接受/拒绝】，或使用【全部接受/全部拒绝】。  
4. 点击【应用修改/保存章节内容】。

**关键按钮文案**  
- 【AI 优化/分析与优化】  
- 【接受】 / 【拒绝】 / 【全部接受】 / 【全部拒绝】  
- 【应用修改】 / 【保存章节内容】

**注意事项（新手筛选策略）**  
- 优先接受：更清晰的动作顺序、更自然对白、更明确因果。  
- 谨慎接受：新增背景设定、过多内心独白、强行改走剧情。  
- 直接拒绝：擅自改名、擅自新增关键设定、推翻既定事实。

## 9. 一键更新人物认知（正文变化后记得同步）

**入口位置**  
章节的【人物认知】视图。

**操作步骤**  
1. 正文优化并保存后，点击【一键更新人物认知】。  
2. 检查是否出现“认知穿越/提前知道”。  
3. 调整后【保存】。

**关键按钮文案**  
- 【一键更新人物认知】  
- 【保存】

**注意事项**  
- 每次大改正文后都建议同步一次，保证后续章节的行为一致性。

## 10. AI 生成伏笔（记录可回收承诺）

**入口位置**  
章节或项目中的【伏笔/伏笔管理】视图。

**操作步骤**  
1. 点击【AI 分析/生成伏笔】。  
2. 查看系统生成的伏笔条目（描述、类型：短期/长期；状态：待回收/已回收；可见性；可能关联人物/回收章节）。  
3. 选择保留的条目并【保存】。

**关键按钮文案**  
- 【AI 分析/生成伏笔】  
- 【保存】

**注意事项**  
- 伏笔是“承诺”；仅保留最能服务主线与人物成长的条目，避免信息噪音。

## 11. 右侧 AI 助手：@ 提问 + 指令模式（重点详解）

### 11.1 @ 提问（上下文检索式）
**入口位置**  
右侧【AI 助手】面板的输入框。

**操作步骤**  
1. 在输入框键入 `@`，选择对象（章节/设定/人物等）。  
2. 输入问题并发送，例如：  
   - “@第十二章：这章冲突是否足够？有什么升级建议？”  
   - “@人物-林曜：动机在第 8 章是否自洽？如何补一段铺垫？”

**关键按钮文案**  
- 输入框  
- 选择对象弹出列表（@ 后）

**注意事项**  
- 提问尽量带“目标”（要结构方案/修改建议/台词备选等），便于直接落地。

### 11.2 指令模式（可执行编辑命令）
**入口位置**  
输入框上方点击【切换到指令模式】。

**操作步骤**  
1. 开启指令模式后，在输入框写“可执行修改”的命令，例如：  
   - “把本章结尾改成反转式钩子，但不改变已发生事件。”  
   - “将人物设定中的‘背景经历’改写得更克制，删掉夸张形容。”  
2. 发送后查看返回的修改结果/应用建议（以系统实际行为为准）。

**关键按钮文案**  
- 【切换到指令模式】

**注意事项**  
- 写清约束：不改人名、不改关键事实、只改某一段、控制字数等。  
- 大改时优先走“对比选择”流程，避免误伤已确认内容。

## 12. 下载：设定集与章节导航栏提供下载

**入口位置**  
- 设定集导航栏右侧的【下载】图标。  
- 章节导航栏右侧的【下载】图标。

**操作步骤**  
1. 点击【下载】图标。  
2. 若弹窗可选范围/格式，按需选择（如当前条目/整分类/当前章/整卷/整本）。  
3. 确认后开始下载，在浏览器下载列表或本地默认下载目录查看文件。

**注意事项**  
- 下载为空时，先检查是否已【保存】最新修改（章纲/正文/设定）。

## 常见排错与小技巧

- 生成结果偏空：在指令里补充“目标/约束/风格/字数”，必要时先完善摘要或设定。  
- 人物跑偏：检查“出场人物选择 + 人物认知”是否同步更新。  
- 节奏拖沓：在章纲里明确“场景序列与冲突推进”，优化时优先选择“节奏张力”。  
- 名词不一致：拒绝任何擅自改名的改动；建立术语表并统一。  
- 反复来回：优先“分卷→路标→章纲→正文→优化→认知→伏笔”的顺序，减少返工。

## 一次完整“验证流程”（建议每次新项目都跑一遍）

1. 登录成功 → 新建小说完成 4 步向导。  
2. 在设定集对任一条目触发【AI 优化】并进行【对比选择接受/拒绝】后【保存】。  
3. 在分卷点击【AI 优化分卷大纲】 → 进入路标页，新增路标并【生成章节】。  
4. 进入某章：先【AI 生成章纲】 → 【初始化人物认知】 → 【选择出场人物】。  
5. 在正文中【AI 辅助写作】生成片段并【保存章节】。  
6. 在章节执行【AI 优化】并在对比界面【接受/拒绝】后【保存章节内容】。  
7. 【一键更新人物认知】并【保存】。  
8. 在章节执行【AI 分析/生成伏笔】并【保存】。  
9. 在右侧【AI 助手】先做一次“@ 提问”，再切到【指令模式】执行一次编辑命令。  
10. 在设定集与章节导航栏各点击一次【下载】确认文件生成。

—

> 祝写作顺利：先跑通闭环，后逐级提质；每次只改动最必要的部分，稳步塑形你的作品。

## 操作流程（独立章节）

**标准工作流总览（建议每次按此顺序执行）**
- 登录：完成【密码/验证码】登录，进入工作台。  
- 新建向导（4 步）：基础信息 → 题材与风格 → 核心设定 → 总纲生成并创建。  
- 设定集：按分类新增/完善条目 → 触发【AI 优化】并在对比界面【接受/拒绝】后保存。  
- 设定集 AI 审查：点击设定集顶部工具栏【AI审查】进入审查面板；对【条目/分类/全设定】执行审查，使用建议卡片中的【复制替换文本】与【去修改】逐条进入设定条目手动修正并保存；必要时在该条目触发【AI 优化】进入对比界面进行【接受/拒绝】；如提供可下载报告。  
- 分卷与路标：在分卷点击【AI 优化分卷大纲】 → 进入【路标】页新增路标并【生成章节】。  
- 章纲：进入章节后【AI 生成章纲】，微调并【保存章纲】。  
- 人物认知：在章节中【初始化人物认知】，检查并【保存】。  
- 出场人物：在章节选择本章【出场人物】，【保存/应用】。  
- 正文写作：点击【AI 辅助写作】，输入具体写作指令，生成后【保存章节】。  
- 章节优化：在章节执行【AI 优化】，在对比界面逐块【接受/拒绝】，最后【保存章节内容】。  
- 更新认知：完成正文大改后，点击【一键更新人物认知】，核验后【保存】。  
- 伏笔生成：在【伏笔管理】点击【AI 分析/生成伏笔】，筛选并【保存】。  
- AI 助手：右侧面板进行【@ 提问】与【指令模式】编辑，谨慎应用改动。  
- 下载：在设定集与章节导航栏点击【下载】图标，选择范围/格式并下载。

**新手提示**
- 每次只改动必要部分；大改后记得更新人物认知并复查伏笔。  
- 任何自动改动都尽量通过“对比界面”逐条确认，保证已确认事实不被覆盖。  
- 建立“术语表/名词库”条目，统一称谓与关键词，减少未来冲突。
